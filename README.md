# 勇者貪食蛇 RPG

一款以勇者隊伍為主題的 HTML5 Canvas 貪食蛇。玩家在網格上操控隊長收集道具、招募弓箭手／法師／騎士，並抵禦持續追擊的敵人。

## 執行方式

1. **直接開啟**：以瀏覽器開啟 `index.html`（支援 `file://` 協議）
2. **本地伺服器**（推薦）：使用任何靜態伺服器（如 VS Code Live Server、Python `http.server`）
3. **控制方式**：
   - **桌面**：使用方向鍵或 WASD 操控隊伍，或使用滑鼠拖曳滑動
   - **行動裝置**：使用滑動手勢控制方向（上下左右滑動）
4. **開始遊戲**：
   - 遊戲載入後顯示主選單
   - 輸入名字（若 localStorage 有記錄則自動帶入）
   - 點擊「開始遊戲」按鈕開始遊戲
   - 遊戲中可點擊按鈕查看排行榜或遊戲說明

## 核心功能

### 遊戲機制
- **網格移動**：隊長在網格上移動，收集道具招募勇者
- **平滑插值**：使用時間基準的線性插值，確保移動均勻流暢
- **合成進化**：同職業同等級勇者數量滿足條件時自動合成升級 (New!)
- **職業系統**：
  - **弓箭手**：遠程射擊，傷害線性成長
  - **法師**：範圍光環傷害，倍數成長，可升級縮放與降速
  - **騎士**：防禦型，受傷觸發無敵，殺怪回血，榮譽號召招募隊員
- **等級系統**：玩家有等級和經驗值，擊殺敵人獲得經驗值升級
- **能力類型限制**：每輪遊戲最多可解鎖 8 種不同的能力類型 (Strategy)
- **障礙物系統**：每升一級生成石頭，有紅色描邊與生成保護期 (New!)

### 輸入控制
- **鍵盤**：方向鍵或 WASD 控制方向
- **觸控**：滑動手勢控制方向（上下左右滑動），顯示軌跡線與箭頭
- **滑鼠**：拖曳滑動控制方向，與觸控體驗一致

### UI 功能
- **主選單**：統一首頁，包含遊戲標題、版本號、名字輸入框
- **排行榜 Modal**：顯示全球前 5 名和今日前 5 名
- **升級選擇**：升級時彈出三選一升級選項，暫停遊戲邏輯
- **死因分析**：Game Over 時顯示具體死因與趣味吐槽
- **測試功能**：遊戲中點擊🔧按鈕可進入測試修改模式（需密碼）

### 技術特色
- **行動裝置優先**：Mobile First 設計，專為觸控操作優化
- **全螢幕相機系統**：Camera 跟隨角色，實現全螢幕遊戲體驗
- **動態特效系統**：
  - 支援多種特效類型 (粒子、光環、閃爍、崩解)
  - 特效繪製具備半徑防護，防止數值錯誤導致崩潰
  - 隊長死亡時有特殊的崩解特效與延遲結束動畫
- **版本管理**：嚴格的語義化版本控制 (Semantic Versioning)
