# 勇者貪食蛇 RPG

一款以勇者隊伍為主題的 HTML5 Canvas 貪食蛇。玩家在網格上操控隊長收集道具、招募弓箭手／法師／騎士，並抵禦持續追擊的敵人。

## 專案結構
| 檔案 | 說明 |
| --- | --- |
| `index.html` | 主頁面與核心常數定義。 |
| `style.css` | UI 與 Canvas 外框樣式。 |
| `script.js` | 遊戲邏輯、動畫迴圈與降級圖形。 |
| `leader.png` ~ `enemy.png` | 角色與道具圖示。 |
| `GDD.md` | 遊戲設計文件 (Game Design Document)。 |
| `CHANGELOG.md` | 更新紀錄。 |
| `player-guide.html` | 玩家導覽手冊。 |

## 執行方式
1. 直接以瀏覽器開啟 `index.html`。
2. 使用方向鍵或 WASD 操控隊伍。
3. Game Over 後按「重新開始」重啟。

## 文件
- 詳細規則與參數：`GDD.md`
- 玩家用圖文介紹：`player-guide.html`
- 版本記錄：`CHANGELOG.md`
